# 3. Non-goals for v1

- **Team / cloud aggregation.** Single-user, single-laptop only.
- **Fuzzy prompt clustering.** Exact-text fingerprinting only. Embedding-based grouping is v2.
- **Non-coding-agent ingestion.** ChatGPT web, Claude.ai web, Gemini web, direct API usage from app code.
- **Cursor / Windsurf / Kiro / Replit Agent.** Data unavailable or undocumented.
- **Conversation transcript rendering.** Sessions link to external transcript viewer.
- **Historical hook events.** Hooks capture forward only; precision available only post-installation.
- **External hosting.** No telemetry, leaderboards, or remote sync. All data stays local unless explicitly exported.
